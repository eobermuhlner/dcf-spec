# DCF v1.0.0

**Release Date:** 2024-12-20

The first stable release of the **Design Concept Format** specification — a text-first, deterministic format for describing UI design systems and application structures.

## Highlights

- **Complete Specification** — 10 layers covering tokens, themes, components, layouts, navigation, screens, flows, rules, and i18n
- **Validation Profiles** — `lite`, `standard`, and `strict` modes for progressive adoption
- **JSON Schemas** — Machine-readable validation for all DCF file types
- **Reference Examples** — Working implementations for each profile level

## What's Included

| Directory | Contents |
|-----------|----------|
| `spec/v1.0.0/` | Full specification document |
| `schema/v1.0.0/` | 11 JSON Schema files |
| `examples/` | Minimal, standard, and strict examples |
| `decisions/` | Architecture Decision Records |
